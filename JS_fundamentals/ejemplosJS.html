EJEMPLO DE ARRAYS

Methods

  • push/pop

    Push sirve para adicional al final del arreglo un nuevo dato. 

    nameArray.push("valor");

    Pop sirve para quitar el ultimo dato del arreglo. 

    nameArray.pop();
    al asignar el metodo a una variable se obtiene el valor retirado

  • shift/unshift

    Unshift sirve para adicional al inicio del arreglo un nuevo dato. 

    nameArray.unshift("valor");

    Shift sirve para quitar el primer dato del arreglo. 

    nameArray.shift();
    al asignar el metodo a una variable se obtiene el valor retirado

  • indexOf

    indexOf sirve para buscar el indice de un item en un arreglo. 

    var amigos = ["Carlos","Cristian","Ana","Jenny","Cristian"]; 
amigos.indexOf("Ana");//2 
amigos.indexOf("Cristian")//1, no 4 por que es el que encuentre primero 

//Si el item no existe retorna -1

  • slice

    Slice Sirve para copiar partes de un arreglo o el arreglo completo si se necesita. 

                                var frutas =["limon","pera","manzana","ceraza","fresa","banano"]; 
var rojas = frutas.slice(2,5)// se especifica donde comienza y termina en el item siguiente al ultimo.
//rojas contiene ["manzana","ceraza","fresa"]
  • FOREACH

    foreach Sirve para recorrer el contenido de los arreglos de una manera facil. 

                                    var frutas =["limon","pera","manzana","ceraza","fresa","banano"]; 
frutas.forEach(function(fruta){// fruta es el placeholder se puede colocar cualquier nombre pero es donde se guardara el contenido.
console.log(fruta);
    })

    foreach puede tener 2 o 3 parametros, el 1 representa el contenido y el segundo representa el indice 

                            var frutas =["limon","pera","manzana","cereza","fresa","banano"]; 
frutas.forEach(function(fruta,i){// fruta es el placeholder se puede colocar cualquier nombre pero es donde se guardara el contenido.
console.log(i + ": "fruta);
    })

    resultado: 0 : limon 1 : pera 2 : manzana 3 : cereza 4 : fresa 5 : banano

  • Reverse

    Ordena el arreglo al reves ej: [1,2,3,4,5] lo vuelve [5,4,3,2,1] 

                                var num = [1,2,3,4,5];
                                num.reverse();
                                //ahora se recorre y se imprime
                                num.forEach(function(numero){
                                    console.log(numero);
                                    });

    Resultado: 5 4 3 2 1
